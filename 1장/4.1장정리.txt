1. mkdir : (make diretory) 새로운 하위 디렉토리를 만든다 

2. rm : (remove) 파일이나 디렉토리를 삭제한다. // 디렉토리를 삭제할때는 -r 옵션을 사용한다.


3. cd .: 지정한 하위 디렉토리로 이동한다.

4. cd .. : 부모 디렉터리로 이동한다.

5. cd ~ : 홈 디렉토리로 이동한다.


6. pwd : (print working diretory) 현재 경로를 표시한다.


7. ls : 현재 디렉토리 안의 내용을 표시한다.

8. ls -l : 현재 디렉토리 안의 파일과 폴더 상세 정보까지 표시한다.

9. ls -a : 현재 디렉토리 안의 숨김 파일과 숨김 폴더도 표시한다.


10. vim 파일명 : 빔 편집기로 파일을 작성하거나 수정한다.

11. cat 파일명 : 텍스트 창의 내용을 터미널에 표시한다.

12. clear : 터미널 창을 깨끗하게 지운다.

13. exit : 터미널 창을 종료한다.
